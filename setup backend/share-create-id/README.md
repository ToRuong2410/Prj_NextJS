### Đăng ký thông tin kích hoạt

Các bước cần làm:

1. Clone project
2. Cài đặt thư viện cần thiết, bằng cách chạy câu lệnh: npm i
3. Chạy dự án bằng câu lệnh: npm start
4. Nếu không có lỗi gì xảy ra, sẽ có message thông báo Done.
